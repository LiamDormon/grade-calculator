# University Grade Calculator

Mostly vibe-coded SPA for calculating uni grades with credit-weighted modules and weighted assignments, and sub-assignments.

## Screenshots
![Screenshot](https://r2.fivemanage.com/lFhBrfG79qJOleqKHoIRN/Screenshot2026-01-18at01-12-16GradeCalculator.png)

## Tech Stack
- React with TypeScript
- Vite/bun build system
- Zustand for state management
- Tailwind CSS for styling
- Shadcn/neobrutalism.dev for UI components

## Building Locally
1. Clone the repository
2. Install dependencies with `bun install`
3. Run the development server with `bun dev` or build with `bun build`
4. Open your browser to `http://localhost:3000`
5. (Optional) import `/dummy_data.json` to get started with pre-filled data
